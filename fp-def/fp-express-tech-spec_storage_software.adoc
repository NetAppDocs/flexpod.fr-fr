FlexPod Express prend en charge les systèmes de stockage qui exécutent le système d'exploitation NetApp ONTAP.



== NetApp ONTAP

Le logiciel NetApp ONTAP est le système d'exploitation qui fonctionne sur les systèmes de stockage AFF, ASA et FAS. ONTAP offre une architecture de stockage hautement évolutive qui permet des opérations sans interruption, des mises à niveau sans interruption et une infrastructure de données agile.

Pour plus d'informations sur ONTAP, consultez le http://www.netapp.com/us/products/platform-os/ontap/index.aspx["Page produit ONTAP"^].
