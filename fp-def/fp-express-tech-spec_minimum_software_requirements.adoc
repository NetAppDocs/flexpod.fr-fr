== Configuration logicielle requise pour FlexPod Express avec NetApp AFF ou FAS

La configuration logicielle requise pour le FlexPod Express avec NetApp FAS comprend :

* ONTAP 9.1 ou version ultérieure
* Cisco NX-OS version 7.0(3)I6(1) ou ultérieure
* Dans la configuration gérée par Cisco UCS, Cisco UCS Manager UCS 4.0(1b)


Tous les logiciels doivent être répertoriés et pris en charge dans le http://mysupport.netapp.com/matrix["NetApp IMT"^]. Certaines fonctionnalités logicielles peuvent nécessiter des versions de code plus récentes que les versions minimales indiquées dans les architectures précédentes.
