== Cisco UCS X-Series Direct

Cisco UCS X-Series Direct est une nouvelle architecture qui intègre une paire de Cisco UCS Fabric Interconnect 9108 100G avec le châssis Cisco UCS X9508 et les nœuds de calcul Cisco UCS X-Series pour fournir la solution UCS X-Series Direct.

Pour un aperçu rapide des capacités et fonctionnalités de Cisco UCS X-Series, consultez link:https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-aag.html["Cisco UCS X-Series Direct Aperçu"^].

Cisco UCS X-Series Direct se compose des éléments suivants :

* Interconnexion de fabric Cisco UCS 9108 100G
* Cisco UCS X9508 Châssis
* Cisco UCS X-Series Nœuds de calcul
* Composants optionnels :
+
** Nœuds PCIe et module X-Fabric
** Deuxième châssis Cisco UCS X9508 avec nœuds de calcul X-Series et modules de structure intelligente Cisco UCS 9108-100G et jusqu'à quatre serveurs rack C-Series





NOTE: Veuillez consulter ce lien pour obtenir les spécifications techniques complètes du UCS X-Series Direct : https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-ds.html["Fiche technique directe Cisco UCS X-Series"].

== Options de serveurs en rack Cisco UCS C-Series

Les serveurs rack Cisco UCS série C sont disponibles en versions 1U et 2U, avec différentes options de processeur, de mémoire et d'E/S. Les références indiquées dans le tableau ci-dessous correspondent aux serveurs de base. Elles n'incluent pas les processeurs, la mémoire, les disques, ni les cartes Peripheral Component Interconnect Express (PCIe). De multiples options de configuration sont disponibles et sont prises en charge dans l'architecture FlexPod.
