= Plateformes FlexPod
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
Il existe deux plateformes FlexPod :

* *FlexPod Datacenter.* Cette plateforme est une infrastructure de centre de données virtuel massivement évolutive adaptée aux applications d'entreprise de charge de travail, à la virtualisation, au VDI et au cloud public et privé. FlexPod Datacenter a ses propres spécifications, qui sont documentées dans https://docs.netapp.com/us-en/flexpod/fp-def/dc-tech-spec_solution_overview.html["Caractéristiques techniques des data centers FlexPod"^].
* *FlexPod Express.* cette plate-forme est une infrastructure convergente compacte conçue pour les bureaux distants et les bureaux distants.


Ce document présente les spécifications techniques de la plateforme FlexPod Express.
