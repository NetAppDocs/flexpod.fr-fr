= Autres exigences
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
Les autres conditions requises pour FlexPod Express sont les suivantes :

* Si vous prévoyez d'utiliser VMware ESXi comme hyperviseur, consultez la link:https://compatibilityguide.broadcom.com/["Guide de compatibilité Broadcom"] pour vérifier que vos composants matériels et logiciels sont pris en charge.
* Pour un déploiement basé sur OpenShift, assurez-vous de disposer des composants matériels et logiciels nécessaires qui sont compatibles avec OpenShift. Consultez la link:https://catalog.redhat.com/en/search["Catalogue OpenShift"] pour plus d'informations.
