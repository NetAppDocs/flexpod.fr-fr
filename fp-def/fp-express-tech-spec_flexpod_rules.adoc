= Règles FlexPod
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
La conception FlexPod prend en charge une architecture d'infrastructure flexible qui englobe une variété de composants matériels et de versions logicielles.

Utilisez les ensembles de règles comme guide pour créer ou assembler une configuration FlexPod valide. Les nombres et les règles indiqués dans ce document représentent les exigences minimales pour FlexPod et peuvent être étendus en utilisant les produits pris en charge au sein des familles de produits incluses, selon les besoins des différents environnements et cas d'utilisation.
